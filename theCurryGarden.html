Background

Out of mentioned projects, this one was by far my favourite. I had to create a booking and ordering system for a restaurant. The project was completed in VB.Net. You were required to complete the project using the waterfall method. I enjoyed it because I had interactions with a real client and the project was fun to complete because I was 1 year into basic coding and the fact that I could achieve this was great. The program had many cool features like automated emails, Real lifetime on when orders are being done , a reward system. All stored with in a database using integrated SQL.

What I learnt

  • Created a fully functioning food booking system incorporating bookings, order updates, a rewards system, inventory checks, financial data.
  • How to provide for different access levels of a company
  • Waterfall methodology
  • Integrated a Email API that allows you to send emails
