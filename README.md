# Tayeeb Islam — Portfolio

Personal portfolio site for [tayeebislam.com](https://tayeebislam.com), built with Astro, React, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

Requires **Node.js 24+** (see `.nvmrc`: run `nvm use`).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` deploy automatically via GitHub Actions to GitHub Pages.

## Stack

- [Astro 7](https://astro.build) — static site generator
- [React 19](https://react.dev) — interactive project filter
- [Tailwind CSS 4](https://tailwindcss.com) — styling via `@tailwindcss/vite`

Requires **Node.js 24+**.
